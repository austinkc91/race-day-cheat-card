## REMAINING WEAKNESSES & FUTURE RECOMMENDATIONS

### Problems NOT solved by improvements 1-4:

1. **Track Condition Sensitivity:** March 7 (sloppy) was a 0% day. No amount of consensus helps when the track is off. RECOMMENDATION: Add a weather/track condition modifier that reduces bet sizes by 50% on off-tracks and shifts strategy toward closers.

2. **Single Source Dependence for Backtest:** We could only get SFTB picks for all 5 days. A true multi-source backtest requires archiving all 6 source picks daily. RECOMMENDATION: Build automated pick scraping that saves all 6 sources' picks to a database before each race day.

3. **Small Sample Size:** 47 races across 5 days is still small. Statistical confidence requires 200+ races (20+ days). RECOMMENDATION: Continue tracking for 2+ months before drawing firm conclusions.

4. **No Live Odds Integration:** Morning line odds ≠ actual tote odds. A horse at 15/1 ML might go off at 8/1. RECOMMENDATION: Use live tote odds for Value Score calculation in the live-update loop (already possible since we search for live odds).

5. **No Jockey/Trainer Angle:** Hot jockeys and trainers at specific meets can predict winners better than algorithms. At Oaklawn, Steve Asmussen dominates. RECOMMENDATION: Add a "Hot Barn" bonus source that tracks trainer win% at the current meet.
